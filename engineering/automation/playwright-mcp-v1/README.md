# Playwright MCP v1

Primera versión de la skill de Playwright MCP.

## Uso

Proporciona instrucciones y mejores prácticas para interactuar con el servidor MCP de Playwright. Controla navegadores reales (Chromium, Firefox, WebKit) desde un LLM mediante el protocolo MCP.

> Ver también: [Playwright MCP v2](../playwright-mcp-v2/) para la versión extendida.
