# Idiomatic Translator

Traductor idiomático español ↔ inglés.

## Uso

Se activa cuando el usuario envía texto para traducir o escribe en español/inglés y necesita la versión en el otro idioma. Preserva el tono idiomático — no es una traducción literal. También responde a frases como "how do I say X in English/Spanish?", "write this in English", "put this in Spanish".
